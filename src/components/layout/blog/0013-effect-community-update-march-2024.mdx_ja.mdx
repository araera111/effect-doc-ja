今月の Effect コミュニティでの出来事をご紹介します！先月の Effect Days カンファレンスの重要な瞬間を共有する喜びを感じておりますが、それだけではありません。それでは、さっそく始めましょう！

## Effect Days カンファレンスの振り返り

私たちは、Effect に完全に捧げられたあの濃密な数日間に共有した経験に対して、感謝と畏敬の念を禁じ得ませんでした。洞察に満ちた講演や魅力的なワークショップでいっぱいでした。

私たちはコミュニティとつながり、アイデアを交換し、物語を共有し、新しい友情を築く素晴らしい時間を過ごしました。数週間前に共有した [Effect Days のハイライト動画](https://www.youtube.com/watch?v=CfbR-1XRlTA&ab_channel=Effect%7CTypeScriptatScale) は、私たちが経験したその日の活気に満ちたエネルギーのほんの一端を垣間見るものです。

多様で才能豊かな個々の Effect 開発者が集まり、それぞれが独自の視点と専門知識を持ち寄ることができたのは、まさに特権でした。

カンファレンスの前日には、[Ethan Niser](https://www.youtube.com/watch?v=3Ii5Ny-FVIg&ab_channel=Effect%7CTypeScriptatScale) と [Maxwell Brown](https://www.youtube.com/watch?v=naI9vNF801w&ab_channel=Effect%7CTypeScriptatScale) が、Effect エコシステムのさまざまなトピックに関する全日制のワークショップを2つ行いました。参加者からの [フィードバック](https://x.com/EffectTS_/status/1770783115788562573?s=20) は非常に好評で、Effect についてさらに学びたいという熱意が感じられます。

私たちにとって、Effect Days は情熱的な人々が集まり、学び、つながり、成長することによって生まれる素晴らしい出来事を思い出させてくれました。

さらに、マリオカートトーナメントや激しいレーザータグセッションの楽しい瞬間を忘れてはいけません。スリリングなバトルの中から数人の勝者が生まれ、一人の名誉ある犠牲者が生まれました。

カンファレンスに参加できなかった方も心配いりません！私たちの YouTube チャンネルで、Johannes Schickling の [基調講演](https://www.youtube.com/watch?v=PxIBWjiv3og&ab_channel=Effect%7CTypeScriptatScale)を視聴できます。今後の動画に関する最新情報を得るために、ぜひチャンネルを購読してください！

## Effect コミュニティ注目人物

[Milad Vafaeifard](https://www.linkedin.com/in/miladvafaeifard/) の貢献と、特に Effect Days でのポジティブなエネルギーをもたらしたことに光を当てたいと思います。カンファレンス後、Milad はウィーンでの自分の経験と Effect に関する旅についての興味深い動画を共有しました。

Milad は自身の [YouTube チャンネル](https://www.youtube.com/@SignLanguageTech) を運営し、Effect を含むソフトウェア開発に関する教育コンテンツを共有しています。最近では、手話で「Effect」のためのユニークなサインも作成しました。

## Effect コミュニティイベント

4月に開催される [Effect パリミートアップ](https://www.meetup.com/fr-FR/effect-paris/) の日程をカレンダーに記入してください。これは、有意義な議論に参加し、知識を深め、フランス語を話す他の Effect 開発者とネットワークを築く機会です。このミートアップは、パリに本社を置く独立系ソフトウェア開発会社 Evryg の Jean-Baptiste Musso と Antoine Coulon が組織しています。ミートアップのスケジュールの最新情報をお待ちください！
